ความสามารถ

ทักษะเบื้องต้นด้านการเว็บไซต์ HTML CSS JAVASCRIPT MYSQL PHP | การออกแบบภายใน การออกแบบบ้าน การตัดต่อ SKETCHUP PHOTOSHOP ILLUSTRATOR | การบริหาร (เจ้าของแบรนด์ RAGER และร้าน KINGBOOSTUP) การกีฬา และเรื่องการเข้าสังคม เพราะผมเป็นคนเฟลนลี่

  • ทักษะเบื้องต้นด้านการเว็บไซต์

    ผมชอบในการออกแบบเว็บไซต์เองตั้งแต่เด็ก อาจจะไม่ได้เขียนเก่ง แต่พร้อมที่จะเรียนรู้ไปเรื่อยๆ และพร้อมรับประสบการณ์ใหม่ๆเพิ่มเติม

  • เจ้าของธุรกิจ

    ผมลองเปิดแบรนด์เสื้อผ้า และเปิดเพจร้าน ทุกแพตฟอร์มรวมมีผู้เข้าถึงและติดตามมากกว่า 50,000 คน อยากหาประสบการณ์ เพราะชอบการบริหารธุรกิจเหมือนกัน

  • ออกแบบ ตัดต่อ

    ผมมีความคิดสร้างสรรค์ด้านนี้ เคยรับจ้างออกแบบบ้าน ออกแบบสวน และรับตัดงานวีดีโอ รูปภาพ

"ผมคือใคร" ?

ผมชื่อฟิวส์ ชื่อจริง นายกิตติพันธุ์ ปานทอง อายุ 18 ปี ชอบความแปลกใหม่ และไม่ซ้ำใคร ทำให้ตัวเองโดดเด่น

— แนะนำตัวเอง

รักในการเขียนเว็บไซต์ การเล่นกีฬา การบริหารงาน ชอบมากๆ ผมเชื่อว่าผมอายุเท่านี้ ผมสามารถต่อยอดหลายๆอย่างที่ผมคิดไปได้อีก ขาดแค่เงินทุนที่พร้อมสนับสนุนผม ในอนาคตอยากเป็นนายตัวเอง มีธุรกิจของตัวเอง

— ความฝัน

ผมมาสมัครค่ายนี้เพื่อปูทางในการเข้าสถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง โดดเด่งเรื่องภาควิชาวิศวกรรมคอมพิวเตอร์ คณะวิศวกรรมศาสตร์ ผมอยากเข้าเรียนที่นี่มากๆ หลักสูตรน่าสนใจ น่าเรียน
